# Embeddable Chat Widget

A modern, embeddable React chat widget with ChatGPT integration.

## Installation

```bash
npm install embeddable-chat-widget
```

## Usage

```javascript
import { ChatWidget } from 'embeddable-chat-widget';

// In your React app
<ChatWidget 
  apiKey="your-openai-api-key"
  theme="light"
  position="bottom-right"
/>
```
